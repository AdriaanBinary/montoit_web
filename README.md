# Montoit Autocomplete

A Nuxt.js frontend for the Montoit API autocomplete endpoint.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to `http://localhost:3000`

## Build for production

```bash
npm run build
npm run preview
```

## Features

- Real-time autocomplete search
- 500ms debounce to optimize API calls
- Minimum 2 characters required to search
- Results show name and type
- Built with Nuxt 3 and Tailwind CSS
